# 玄默 - 命理玄学工具网站

一个沉浸式的命理探索空间，提供塔罗牌占卜、六爻起卦、星盘排盘、星座分析、八字排盘等玄学测算服务。

## 技术栈

### 前端
- React 18 + TypeScript 5
- Vite 5
- Tailwind CSS 3
- React Router 6
- Zustand（状态管理）
- React Hook Form + Zod（表单验证）
- Framer Motion（动画）
- Lucide React（图标）

### 后端
- Node.js 18
- Express 4
- Prisma 5
- PostgreSQL（数据库）
- Zod（数据验证）

## 核心功能

### 1. 塔罗占卜
- 支持用户手动输入 1-78 的数字选择卡牌
- 支持单牌、三牌、凯尔特十字等牌阵
- 提供详细的卡牌解读和综合分析

### 2. 六爻起卦
- 自动生成卦象
- 展示本卦、变卦、动爻
- 提供详细的卦辞解读

### 3. 星盘排盘
- 支持输入出生日期、时间、地点
- 支持合盘分析
- 展示太阳、月亮、上升星座等基本信息
- 提供性格、事业、感情分析

### 4. 星座分析
- 支持所有 12 个星座
- 展示性格、运势、事业、感情等分析
- 提供星座匹配度分析

### 5. 八字排盘
- 支持输入出生日期、时间、性别
- 生成完整的八字命盘
- 展示五行、十神等信息
- 提供性格、事业、感情、大运分析

### 6. 统一结果页
- 支持复制结果
- 支持分享结果
- 支持重新测算
- 提供详细的结果展示

## 快速开始

### 前端开发

1. 进入前端目录
```bash
cd frontend
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

### 后端开发

1. 进入后端目录
```bash
cd backend
```

2. 安装依赖
```bash
npm install
```

3. 生成 Prisma 客户端
```bash
npm run prisma:generate
```

4. 启动开发服务器
```bash
npm run dev
```

## 生产部署

### 前端构建
```bash
cd frontend
npm run build
```

### 后端构建
```bash
cd backend
npm run build
```
